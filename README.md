# thoth
Wordlist Generator
## How to Use:
* python tooth_1.2.py
* Enter your minimum lenght
* Enter your maximum lenght
* Choose the Option
* Start
  -- The files will save to the 'books' folder with 10000000 iterations
  -- 10000000 words for each .txt file
* Ctrl+C to cancel process
